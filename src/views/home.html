welkom op de home pagina!


Deze website is geheel gebouwd met mijn eigen framework!

De website werkt wordt gehost op een virtual host, met een .htaccess file die alle pagina verzoeken terugstuurt naar index.php. Index php instantieert een router object, die de url die de gebruiker heeft opgegeven analyseert, om vervolgens de juist controller en methode te laden uit de OOP klassenstructuur. Composer word gebruikt om de dependecies te regelen. Twig is de template engine waarmee html (views) gerendered worden waar nodig. RedbeanPHP is de Object Relational Manager, die mij een makkelijke wrapper geeft rondom pdo voor het maken van database structuren.


De login gegevens van de test gebruiker zijn opgeslagen in B_CRYPT gehashte vorm. Een login poging vraagt de hash op uit de database doormiddel van RedBean. Als de hash van het ingevulde wachtwoord overeenkomt met de "true" hash uit de database, wordt de login gevalideert en een sessie token aangemaakt, met een base64 encoded string van de cryptografisch robuuste random_bytes(). Er worden verbose waarschuwing gegeven, en niet-ingelogde gebruikers worden geweerd.
